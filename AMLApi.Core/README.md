# AML API

**AML API** is a C# project providing an interface for accessing and analyzing data from the **AML (All Mode List)** — a ranked list of the most difficult *Five Nights at Freddy’s* max modes.  

## Installation

All builds avaible through nuget package
- [AMLApi.Core](https://www.nuget.org/packages/AMLApi.Core/)

Or by .NET CLI:
- `dotnet add package AMLApi.Core`
